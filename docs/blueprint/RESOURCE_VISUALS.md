# Resource Visuals

## Purpose
This file defines the visual role of the core resources.

## Primary Resource: Main Coin

Reference asset:
- `assets/main-coin-reference.png`

Working role:
- primary currency
- main progression resource
- strongest resource in the player top bar
- visual anchor for the economy

## Visual DNA
The coin defines a key part of the `PLAYER` visual identity.

Core traits:
- circular medallion shape
- gold outer rim
- dark inner ring
- emerald / cyan glowing core
- small square light nodes around the edge
- organic plant symbol in the center
- tech-lab precision with bio-growth meaning

## Color Direction

Primary coin colors:
- gold rim: `#c7a22b`
- dark green glass: `#062b26`
- emerald glow: `#42f0b3`
- cyan highlights: `#57e5da`
- deep shadow: `#050b0d`

## Product Meaning
This coin should not feel like a generic game coin.

It should feel like:
- bio-credit
- lab-grown currency
- energy-backed token
- a resource created by living systems

Avoid:
- flat yellow coin
- cartoon currency
- generic crypto token styling
- overusing blockchain/NFT language in phase 1

## UI Usage

### Top Bar
The main resource counter should use this coin as the strongest icon.

Visual treatment:
- larger than secondary resources
- subtle glow
- small rotating/light pulse animation
- readable amount next to it

### Rewards
When the player earns the primary resource:
- small coin burst
- emerald glow pulse
- soft gold rim flash

### Reactor/Capsule Link
The coin's visual language should influence:
- reactor core glow
- capsule successful harvest glow
- primary economy effects

## Implementation Notes
Phase 1 can use the reference PNG directly.

Later versions may create:
- CSS/SVG coin icon
- lightweight animated coin
- Three.js coin for premium reward moments
- sprite sheet for coin burst animations

## Naming
Final name is not locked yet.

Working names:
- `Bio Credit`
- `Lab Coin`
- `Core Coin`
- `Energy Coin`
- `BC`

Current code may still use older labels like `credits`.
