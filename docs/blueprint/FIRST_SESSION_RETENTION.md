# First Session Retention

## Purpose
The first player session must teach the loop quickly and create curiosity.

The player should not wait too long before the first result.

Goal:

```text
plant fast -> see capsule react -> harvest quickly -> see rare potential -> understand next goal
```

## First 3-5 Minutes

### Target Experience
Within the first session the player should:
- plant the first starter specimen
- see the capsule visually react immediately
- watch the capsule become alive
- collect the first result
- see the collapse state
- earn Bio Credits
- see that better outcomes exist
- understand the next unlock target

## First Session Flow

```text
open app
receive free starter resources
tap first capsule
choose Basil starter
plant
see instant visual reaction
wait 20-45 seconds
collect
see collapse result
earn Bio Credits
see next goal: unlock Capsule 2 / upgrade / better plant
```

## Starter Timing
Starter plants should be fast.

Suggested first tuning:

| Tier | Example | Time | Purpose |
| --- | --- | ---: | --- |
| Tutorial | Basil Trial | 20-45 sec | teach loop immediately |
| Starter | Basil / Rukola | 1-3 min | early repeat loop |
| Growth | Sunflower / Mint | 5-15 min | first meaningful wait |
| Rare Bio | Chroma / Neon Fern | 30-90 min | longer-term goal |
| Premium/Event | Quantum Bloom | variable | special sessions |

## First Plant
Recommended:

```text
Basil Trial
```

Traits:
- free or nearly free
- very short duration
- low waste
- stable state profile
- small but satisfying reward

Possible first result should still show rare states exist.

Example:
- result: `first`
- reward: `+22 Bio Credits`
- next hint: `Excited and Quantum states can create strains.`

## First Hook
After first harvest, show one clear next goal.

Good options:
- unlock Capsule 2
- upgrade Capsule Lens
- try Rukola for better rare chance
- complete a Calm ritual
- start first Lab synthesis after first strain/result

Do not show all goals at once.

Recommended first hook:

```text
Unlock Capsule 2 with Bio Credits
```

Why:
- simple
- visible progress
- teaches currency value
- leads to monetization later through slots

Later in first-day flow:
- introduce Lab synthesis as the place where capsule outputs become unique items
- introduce Meditation artifacts after the player has at least one Lab-created item

## Progression Hooks

### Hook 1: Better Plants
Show locked higher-tier plants early.

Player sees:
- Chroma locked
- Quantum Bloom locked
- unlock requirement

Effect:
- creates curiosity
- shows depth without explaining too much

### Hook 2: Better States
After every harvest, show the collapse state.

Even if player gets `first`, show the full ladder:

```text
Ground -> First -> Excited -> Quantum
```

Effect:
- makes player want stronger outcomes
- supports upgrades
- supports premium capsule

### Hook 3: Extra Capsules
Capsules are visual progression.

Slot model:
- Capsule 1: free
- Capsule 2: early coin target
- Capsule 3: deeper coin target
- Capsule 4: Stars premium unique capsule

Effect:
- clear expansion goal
- easy monetization surface

### Hook 4: Calm Energy
Meditation should appear as a stabilizer.

Message:

```text
Calm Energy makes rare rituals cleaner and lowers waste.
```

Effect:
- gives meditation a real purpose
- makes product feel more meaningful

### Hook 5: Premium Uniqueness
Stars capsule should promise unique quality, not only speed.

Possible promise:

```text
Quantum Capsule guarantees a unique mutation category.
```

Effect:
- premium offer feels special
- avoids pure pay-to-win framing

## Monetization Path

The first monetization path should be:

```text
free capsule -> coin capsule -> premium unique capsule
```

This is easier to understand than selling many random boosts.

Phase 1 Stars offer:
- `Quantum Capsule`
- unlocks Capsule 4
- includes unique visual capsule skin
- unlocks guaranteed unique mutation category

Avoid in first MVP:
- many consumable boosts
- confusing bundles
- marketplace

## Reward Design

First rewards should feel generous but bounded.

Example first-session grants:

```text
starting Bio Credits: 40
starting Biomass: 20
starting Calm: 0-5
first Basil Trial cost: 0-5
first reward: 20-30 Bio Credits
Capsule 2 unlock target: 80-120 Bio Credits
```

This lets the player see progress but not unlock everything instantly.

## Visual First Session
The first capsule should visually change quickly:
- preview changes when selecting a plant
- fill animation after planting
- plant-specific pattern appears
- seed glow
- fluid fills
- soft particle drift
- gold rim pulse near ready state
- collapse flash

First harvest should include:
- coin glow
- state reveal
- next goal card

The first wow moment must happen before harvest.

Minimum:
```text
select Basil -> capsule previews emerald leaf-vein pattern -> plant -> capsule fills and starts pulsing
```

## Success Criteria
First session succeeds if the player can say:

```text
I planted something.
It transformed.
I got a result.
There are rarer results.
I know what to do next.
```
