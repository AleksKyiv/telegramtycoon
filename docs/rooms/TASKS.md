# Tasks Room Passport

## Role

Tasks give starter energy and guide users toward external channels: Telegram channel, YouTube, TikTok, Instagram, and campaign actions.

## Current Elements

- Tasks room exists in bottom navigation
- Bonus action concept

## Future Direction

Tasks should solve early-game friction: the user has little energy at the start, so simple actions unlock first movement.

## Stable Rules

- Rewards must not be abusable.
- External actions should be easy to understand visually.
- Verification can start simple, but paid/reward-heavy tasks need stronger checks later.

## Data To Save

- Task viewed
- Task clicked
- Task claimed
- Reward amount
- Verification status

## Admin Needs

- Which tasks are clicked
- Which rewards are claimed
- User conversion by campaign
- Fraud/duplicate checks later
