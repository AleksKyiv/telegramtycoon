# Zen Room Passport

## Role

Zen is the core product. The game exists to lead users into meditation through curiosity, progression, and reward.

## Current Elements

- Zen room prototype
- Visual meditation surface
- Sound concept
- Zen energy concept

## Future Direction

Zen should feel premium enough that a user wants to stay for 5 minutes. It should include audio, breathing rhythm, visual calm, artifact influence, and meaningful rewards.

## Stable Rules

- Zen is not just another room. It is the main value.
- Sound and visuals must be calm, not noisy.
- Rewards should encourage completion, not only clicking.
- Session completion must be server-recorded.

## Data To Save

- Session started
- Session completed
- Duration
- Zen energy earned
- Artifact modifiers used
- Streaks

## Admin Needs

- Zen opens
- Starts
- Completions
- Average session time
- Drop-off time
- Zen energy generated
