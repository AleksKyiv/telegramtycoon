## Що треба налаштувати на Render

У Render треба додати змінні:

```text
ADMIN_PASSWORD=твій-пароль-для-входу-в-адмінку
ADMIN_COOKIE_SECRET=довгий-випадковий-текст-для-захисту-сесії
```

`ADMIN_PASSWORD` - це пароль, який ти вводиш на сторінці `/admin`.

`ADMIN_COOKIE_SECRET` - це технічний секрет для захисту входу. Його не треба запам'ятовувати.

## Важливо

Не пиши реальний пароль в GitHub і не вставляй його в код.

Пароль має жити тільки в Render у розділі Environment.
